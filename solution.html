The St. Petersburg Paradox

We use H and T to denote heads and tails respectively:

If you won $2, you would have flipped T, so there is a 1 2 chance.

Similarly, if you won $4, you would have flipped HT, so there is a 1 4 chance.

If you instead won $8, you would have flipped HHT, so there is a 1 8 chance.

And so on forever.

So, what's the expected value?

That would be 2 · 1 2 + 4 · 1 4 + 8 · 1 8 + ⋯ .

Simplifying, it would be 1 + 1 + 1 + ⋯ , which goes to ∞ .
